FAQ

Common questions

Is Blanq official / from OpenAI?

No. Blanq is an independent third-party tool. It is not affiliated with, endorsed by, or sponsored by OpenAI. "ChatGPT" and "GPT" are trademarks of OpenAI.

Is this safe? Could my account get banned?

Blanq only performs actions you can already perform in the normal ChatGPT UI — delete, unarchive, unpin — just in bulk. It uses your own logged-in session, throttles requests to avoid flagging, and asks for confirmation before every bulk action.

Does it collect my conversations?

No. Blanq never reads chat content. It only fetches the list of conversation titles and timestamps that ChatGPT's own sidebar already loads. Nothing leaves your browser.

Can I recover deleted chats?

No — deletion is permanent, just like ChatGPT's own delete button. That's why Blanq always asks for confirmation first. If you're nervous, start by archiving or use the "Older than 365 days" preset.

Will it work if OpenAI changes something?

Blanq uses ChatGPT's own API endpoints. If OpenAI restructures them, Blanq may temporarily break. Updates are pushed through the Chrome Web Store when that happens.

Is it free?

Yes, completely. No paid tier, no trials, no hidden costs. If Blanq saved you time, a Chrome Web Store review is the best way to say thanks.
